
# Fintech Payment Gateway Frontend

Modern payment gateway checkout UI built with React, Vite and TailwindCSS.

## Features

- Card payment with Luhn validation
- Card number auto formatting
- UPI QR code payment
- 3D Secure authentication simulation
- Payment processing spinner
- Receipt page
- Mobile responsive design

## Tech Stack

- React
- Vite
- TailwindCSS
- Axios

## Run Project

Install dependencies

npm install

Run development server

npm run dev

Open browser

http://localhost:5173
